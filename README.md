## 📌 Project Overview
This project focuses on analyzing restaurant and customer order data from Zomato to uncover meaningful business insights. The analysis helps understand customer ordering behavior, restaurant performance, and rating trends.

The project demonstrates real-world data analysis workflow including data cleaning, exploratory data analysis (EDA), and data visualization.

---

## 🎯 Objectives
- Identify popular restaurant categories
- Analyze customer rating patterns
- Compare online and offline ordering trends
- Study average spending behavior
- Extract insights for business decision-making

---

## 📊 Key Insights Explored
✔ Most popular restaurant types  
✔ Distribution of customer ratings  
✔ Online vs Offline order comparison  
✔ Voting and engagement trends  
✔ Cost analysis for different restaurant categories
